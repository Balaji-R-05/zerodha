# Zerodha

A fully responsive clone of Zerodha’s landing page to replicate the user interface and improve frontend development skills. Focused on pixel-perfect layout, responsive design principles, and clean, maintainable code architecture.

<img src="./assets/demo.png" alt="My Zerodha" style="border: 2px solid black">

## Tech Stack

- HTML
- CSS
- Javascript

## Key Features:

- Recreated Zerodha’s website with accurate design and layout with slight improvements.
- Implemented responsive design using Flexbox and CSS Grid for mobile compatibility.
- Added smooth hover effects and transitions for a professional, modern look.
- Optimized structure for fast loading and semantic HTML accessibility.
